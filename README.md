# FDRMenu


1. FDReserve OS for French-digital-rising [https://www.fdreserve.com/](https://www.fdreserve.com/)
2. Clone the repo
   ```sh
   git clone https://github.com/Satcomx00-x00/FDRMenu.git
   ```
   
3. Install NPM packages
   ```sh
   pip3 install -r requirements.txt
   ```
4. To launch
   ```Python
   python3 Menu.py
   ```
